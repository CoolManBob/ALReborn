핫키 리스트.

* 모드변경.
1	- Geometry		모드로 변경
2	- Tile			모드로 변경
3	- Object		모드로 변경
4	- Subdivision	모드로 변경

* 저장/로딩
ALT+S	- Save All
C		- Create Sector with Default Setting
ALT+C	- Create Sector with Detail Setting

* Edit..
CTRL+Z	- Undo ( Geometry , Tile, Object 에만 적용.. ) ( 참고 : 5단계 )
CTRL+Y	- Redo ( Geometry , Tile, Object 에만 적용.. ) ( 참고 : 5단계 )

* 카메라 워킹
W		- Forward
S		- Backward
A		- Step Left
D		- Step Right
SPACE	- Lift Up
Z		- Lift Down
Q		- Rotate CW
E		- Rotate CCW
( SHIFT와 조합시 가속 가능 )
CTRL+R	- Camera Reset ( 지형 Dialog 띄움 )
TAB		- 현재 섹터 상위로 이동.
ALT 홀드 - 현재 섹터를 변경하지 않음.

CTRL + F1/F2/F3/F4	- 현재 카메라 위치 저장
F1/F2/F3/F4			- 저장됀 위치로 카메라 이동.
Ctrl + 1			- 1인칭 시점이동

* 표시 변경
G		- Grid On/Off

* Geometry 모드
SHIFT + 지형 조절	- 가속기능.
ALT + 클릭			- 지형 평평하게 만들기.

* Tile 모드
I					- 바닥 타일을 선택하고 , 바닥 찍기 모드로 이동. ( 다른모드에서도 작동 )
O					- 알파타일을 선택하고 , 위에 얺기 모드로 변경 ( 다른모드에서도 작동 )
(배경그리기 모드에서..)
CTRL + 드래그		- 선택한 타일을 범위 영역에 채워 넣음.
(알파 타일링 모드 )
CTRL + 타일 클릭	- 해당 타일 주위로 알파를 먹임.
F					- 필기능.

[ ]					- 브러시 크기 조절
ALT + 클릭			- 이전 방식의 타일 찍기.

* Object
DEL		- 선택 오브젝트 삭제
ALT 누르고 있기 - 선택됀 오브젝트 변경돼지 않음.

* Subdivision
DEL		- 선택 서브디비젼 영역 삭제.
M		- 선택영역 타일 통맵으로 합침.
